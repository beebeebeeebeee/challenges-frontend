# Challenges Frontend

# Get Stated

## install dep.
```
yarn
```
if you not using yarn, me recommand you to install yarn.<br>
using *npm* is also ok
    ```
    npm install
    ```

## all set! start the server
```
yarn start
```
or
```
npm start
```

## access
service is started at http://localhost:3000/ by default
